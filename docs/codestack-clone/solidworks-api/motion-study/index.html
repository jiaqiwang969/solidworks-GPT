Automating Motion Study using SOLIDWORKS API

{ width=250 }

SOLIDWORKS Motion Study API provides specific interfaces in the separate SwMotionStudy library. It is required to explicitly add the reference to this library to your application if early binding is needed to be used.

Base interface IMotionStudyManager can be accessed via IModelDocExtension::GetMotionStudyManager SOLIDWORKS API method.

This section contains macros and code examples for automating Motion Studies in SOLIDWORKS using API.
