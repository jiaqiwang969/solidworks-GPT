本节包含了SOLIDWORKS插件、宏和独立应用的部署指南和示例，以供用户在其机器上进行部署。

可以通过[手动方式](manual)进行部署。在这种情况下，所有的二进制文件和注册过程都需要手动完成，或者通过命令行脚本完成。

通过[msi安装程序](installer)进行部署可以提供更好的用户体验，因为客户端将通过安装过程进行引导。此外，升级和卸载的过程由Windows控制面板控制。
